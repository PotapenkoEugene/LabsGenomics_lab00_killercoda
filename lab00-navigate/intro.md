# Navigate the Server

You are on `controlplane` — your home computer.

The genomics server (`labserver`) is organized, but nothing is where you expect. Files are buried in deep directories, and there are hundreds of them. Typing full paths by hand would take forever.

In this exercise you will:

1. Use **Tab completion** to navigate a deeply nested directory without typing the full path
2. Use **`ls` flags** to find the largest and most recently modified file among hundreds
3. Use **`mkdir -p`** to create your own project workspace in one command
