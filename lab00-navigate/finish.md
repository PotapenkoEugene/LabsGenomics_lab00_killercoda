# Well done

## What you practiced

| Command | Purpose |
|---------|---------|
| `Tab` | Complete a filename or path — press twice to see options |
| `cd path` | Change into a directory |
| `cd ~` | Go to your home directory |
| `pwd` | Print your current location |
| `ls` | List files in the current directory |
| `ls -lh` | Long format with human-readable sizes |
| `ls -lhS` | Sorted by size — largest first |
| `ls -lt` | Sorted by modification time — newest first |
| `ls -a` | Show hidden files (names starting with `.`) |
| `\| head` | Show only the first 10 lines of output |
| `cp SOURCE DEST` | Copy a file |
| `mkdir -p a/b/c` | Create a directory tree in one command |
