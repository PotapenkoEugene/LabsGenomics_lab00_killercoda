# Welcome to the Server

A single Illumina sequencing run produces **15–100 GB of data**.

That is more than most laptops have in free RAM. So genomics is not done on laptops — it is done on remote servers that run 24/7 and have the RAM and disk to match.

You are now connected to one of those servers. In this lab you will learn your first moves:

- Who you are on this machine (`whoami`)
- Where you are on this machine (`pwd`)
- How to SSH to a **second machine** — and verify that your identity and location both changed

These two commands — `whoami` and `pwd` — are the first thing any bioinformatician runs after landing on a new server. Memorize them.

> **Two terminals are available** — one for each machine (tabs at the top). You will use both.
